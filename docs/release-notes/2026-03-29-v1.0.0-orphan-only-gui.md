# v1.0.0 - Orphan-Only GUI

本版本把 CodexSubMcp GUI 正式收口到更安全、更容易理解的 orphan-only 主流程。

## 主要更新

- GUI 主路径收口为：
  - 刷新
  - 父进程识别校验
  - orphan 结果生成
  - 执行清理
- 不再把旧的 stale cleanup 产品路径暴露为 GUI 主能力
- 总览页内聚：
  - 识别状态
  - orphan 清理结果
  - MCP inventory
- 左侧导航进一步压缩为：
  - 总览
  - 计划任务
  - 配置
  - 日志
- 清理动作保持管理员提权链路
- MCP 结果仍支持复制与导出，但不再单独占用一个导航页
