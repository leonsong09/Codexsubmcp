# v0.2.2 - Packaging Hotfix

本版本是 `v0.2.0` 的连续热修，目标是把 Windows `exe` 打包、源码安装和 GitHub Actions 发布链路补齐到可重复发布状态。

## 主要更新

- 修复 `pip install -e .[dev]` 的 setuptools 自动包发现问题
- 显式限制仅打包 `codexsubmcp*` 包，排除 `packaging*` 和 `temp*`
- 新增打包元数据回归测试，避免后续再次把发布链路改坏
- 修复 legacy setup 脚本在非 UTF-8 Windows 代码页下输出中文帮助文本时的 `UnicodeEncodeError`
- 重新构建 `CodexSubMcpManager.exe` 并生成 `SHA256` 校验文件

## 验证

- `QT_QPA_PLATFORM=offscreen venv\Scripts\python.exe -m pytest tests/test_gui_smoke.py -q`
- `QT_QPA_PLATFORM=offscreen venv\Scripts\python.exe -m pytest -q`
