# v0.2.3 - Startup Fix

本版本修复了 Windows 单文件 GUI 发布包的默认启动行为。

## 主要更新

- 修复 `CodexSubMcpManager.exe` 无参数启动时直接退出的问题
- 现在双击 `CodexSubMcpManager.exe` 会默认进入 GUI，而不是走 CLI help 分支
- 新增 CLI 回归测试，确保无参数启动默认调用 `launch_gui()`

## 验证

- `venv\Scripts\python.exe -m pytest tests/test_cli.py::test_main_without_args_launches_gui -q`
- `QT_QPA_PLATFORM=offscreen venv\Scripts\python.exe -m pytest -q`
- `venv\Scripts\pyinstaller.exe packaging/windows/CodexSubMcpManager.spec --noconfirm`
- 无参数启动 `dist\CodexSubMcpManager.exe`，5 秒内进程保持存活

## 发布产物

- `CodexSubMcpManager.exe`
- `checksums.txt`

## 已知说明

- PyInstaller warning 文件中仍会出现若干 Windows 下无关的 POSIX 可选模块缺失记录，这不影响当前 GUI 发布
